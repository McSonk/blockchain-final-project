Getting Started with Ethereum

1. Create a new account

In this first step, we need to create a new account for your Ethereum node. For creating your account, you'll need a password that you will use to lock or unlock your account when initiating a transaction. Please enter your password below and click generate a new account to get started.

Please make a note of your generated account address before moving to the next step.

2. Generate the Genesis file for your Ethereum chain

Enter the contents for your custom Genesis file in the box below. The content here will be used to initialize your Ethereum node in the next step.

3. Initialize your Ethereum Node

In this step, we will initialize your Ethereum node using the custom Genesis file that you created in the previous step. If you did not create a file, please complete step 2 before performing step 3.

4. Start your Ethereum Node

Finally, we will start our Ethereum node to be able to join the Blockchain and transact on it. To do so, simply click the button below and then wait for the status to change. Then, move to the next step.
